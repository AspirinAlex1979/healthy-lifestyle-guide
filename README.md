# 🥗 Проєкт "Перезавантаження": Посібник зі здорового тіла

Цей посібник створений для системного підходу до схуднення та покращення фізичної форми в домашніх умовах або на вулиці. План розрахований на **30 днів**.

---

## 🍏 Частина 1: Стратегія харчування (Схуднення без стресу)

Головна мета — створити невеликий дефіцит калорій та збалансувати білки, жири та вуглеводи.

### 📸 Приклади та візуалізація харчування
1. **Метод "Здорової тарілки":** [Переглянути фото-приклад](https://maminaferma.com.ua/wp-content/uploads/2021/11/zdorova-tarilka-ukr.jpg) — це основний візуальний орієнтир для кожного прийому їжі.
2. **Ідеї корисних сніданків:** [Переглянути добірку фото](https://www.pinterest.com/search/manual/?q=healthy%20breakfast%20ideas)
3. **Підготовка їжі на тиждень (Meal Prep):** [Приклад порцій на фото](https://www.pinterest.com/search/manual/?q=healthy%20meal%20prep%20containers)

### 📅 Приклад меню на день
- **Сніданок:** Омелет з овочами + цільнозерновий хліб.
- **Обід:** Запечене м'ясо/риба + гречка/кіноа + велика порція салату.
- **Вечеря:** Легкий білок (біла риба, сир або курка) + зелені овочі.
